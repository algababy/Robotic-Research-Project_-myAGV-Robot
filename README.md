# Robotic-Research-Project_-myAGV-Robot
## myAGV Robot Mapping using LiDAR and ROS

This project demonstrates environment mapping and robot navigation using the **myAGV PI 2023 mobile robot platform**.

The robot uses **LiDAR sensors and SLAM algorithms** to scan the environment and generate a map while the robot moves. The mapping process is visualized using **RViz**, and the robot is controlled remotely through a network connection. 
## Technologies
- Python
- ROS (Robot Operating System)
- LiDAR sensor
- RViz
- Raspberry Pi
- Linux

## Key Features
- LiDAR-based environment scanning
- SLAM mapping
- Remote robot control
- Visualization of robot position and map
